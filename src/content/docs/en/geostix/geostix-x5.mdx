The GEOSTIX-X5 is based on Septentrio's [mosaic-X5](https://www.septentrio.com/en/products/gnss-receivers/gnss-receiver-modules/mosaic-x5) GNSS module. It stands out from other GEOSTIX models with several additional features:

## Web configuration interface

**Ethernet via USB:**
when the GEOSTIX is connected to a PC via the magnetic USB cable, it provides access to the module's settings through a simple web browser at [192.168.3.1](http://192.168.3.1/).

<img src="/images/geostix-x5/interfaceSeptentrio.png" style="display: block; margin: auto;" />

## Accessing recorded data

The GEOSTIX-X5 features a built-in SD card that automatically records RINEX data at 1 Hz on a daily basis.

When connected to a PC via the magnetic USB cable, the SD card is exposed as a mass storage device. It contains the directory **SSN/GRB0051** (GRB0051 is the mosaic-X5 code name).

**Directory structure:**
Sessions are stored in subdirectories named **YYDDD**, where:
- YY — year of the first recording
- DDD — day of the year of the first recording

**File naming convention:**
Files follow the pattern **gstxDDDX.YYo** (observations) or **gstxDDDX.YYp** (ephemerides), where:
- DDD and YY — day and year of the first recording
- X — session identifier within the day:
  - `0` for a full-day recording
  - `1`–`9` then `A`–`O` for hourly sessions (hours 1 through 24)

## Configure automatic RINEX saving

<Aside type="tip" title="Default settings">
  By default, the GEOSTIX-X5 saves RINEX data daily with a 1 Hz frequency.
</Aside>

It is entirely possible to change different parameters concerning RINEX recording.
This is done from the web interface, using a browser (address 192.168.3.1)

<Steps>
1. **Click on the "Admin" tab then select "Expert Control"**
    
    
    <img src="/images/geostix-x5/adminExpertControl.png" width="400" style="display: block; margin: auto;" />
   
2. **Click on Logging**
    <img src="/images/geostix-x5/log.png" width="400" style="display: block; margin: auto;" />    

3. **Click on "Internal RINEX Logging"**
    <img src="/images/geostix-x5/internalRinex.png" width="400" style="display: block; margin: auto;" />    

4. **Click on "RINEX Logging Options"**
    <img src="/images/geostix-x5/rinexLog.png" width="400" style="display: block; margin: auto;" />    

5. **Here is the RINEX saving configuration interface:**
    You can then configure hourly or daily saving, frequency, or the RINEX format version. 
    <img src="/images/geostix-x5/confRinex.png" style="display: block; margin: auto;" />    
</Steps>


<Aside type="danger" title="Save the config">
    
    Parameter modifications remain effective **only until** the next GEOSTIX restart.

    To apply the modifications permanently, you need to click on the Save button displayed at the bottom right of the screen.


    <img src="/images/geostix-x5/save.png" width="300" style="display: block; margin: auto;" />    
</Aside>
