title: Quick Start Guide

---

GEOBSYS has developed the GEOSTIX APP to make high-precision GNSS technologies accessible and easy to use for everyone. This guide walks you through the different steps to get started.

To use the GEOSTIX, please follow these steps:

import { Steps } from '@astrojs/starlight/components';

<Steps>

1. **Power on the GEOSTIX.**
    
    by pressing the button located at the base of the GEOSTIX.
    <img src="/images/quick_guide/etapePowerOnSmall.png" width="150" style="display: block; margin: auto;" />
   
2. **Download the GEOSTIX APP on your Android or iOS smartphone.**
    <div style="display: flex; justify-content: center; gap: 2rem;">
        <a href="https://apps.apple.com/ch/app/geostix/id6711349396?l=fr-FR" target="_blank" rel="noopener noreferrer">

        </a>
        <a href="https://play.google.com/store/apps/details?id=app.geostix" target="_blank" rel="noopener noreferrer">
            <img src="/images/quick_guide/playstore.png" width="150" />
        </a>
    </div>

3. **Launch the GEOSTIX APP.**

    Tap the GEOSTIX icon at the bottom left. Select the GEOSTIX device that just appeared: it is now connected to your smartphone.
    <div style="display: flex; justify-content: center; gap: 2rem;">
        <img src="/images/quick_guide/before_connect.png" width="300" />
    </div>
    <div style="display: flex; justify-content: center; gap: 2rem;">

    <div style="display: flex; justify-content: center; gap: 2rem;">
        <img src="/images/quick_guide/connected.png" width="300" />
    </div>

4. **Connect to an NTRIP correction server.**

    Tap "NTRIP setup" and enter the correction server connection parameters.  
    You can, for example, connect to [Centip√®de](https://docs.centipede.fr/)
    <div style="display: flex; justify-content: center; gap: 2rem;">
        <img src="/images/quick_guide/centipede_near.png" width="300" />
    </div>

    Tap "connect"

5. **Start surveying.**

    Return to the map screen.

    Tap the point survey icon at the bottom right, then start an averaged point survey.
    <div style="display: flex; justify-content: center; gap: 2rem;">
        <img src="/images/quick_guide/rdy2survey.png" width="300" />
    </div>
    <div style="display: flex; justify-content: center; gap: 2rem;">
        <img src="/images/quick_guide/averaging.png" width="300" />
    </div>

    You can then tap on the recorded point (on the map) and add a description or a photo.
