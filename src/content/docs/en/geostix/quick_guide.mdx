GEOBSYS a développé GEOSTIX APP pour rendre les technologies GNSS de haute précision accessibles et faciles à utiliser par tous. Ce guide vous accompagne dans les différentes étapes de prise en main.

Pour utiliser le GEOSTIX, veuillez suivre les étapes suivantes :



import { Steps } from '@astrojs/starlight/components';

<Steps>

1. Allumez le GEOSTIX.
en pressant le bouton situé au niveau de l'embase du GEOSTIX


2. Téléchargez l'application GEOSTIX APP sur votre smartphone Android ou iOS.

3. Connectez-vous au GEOSTIX.

4. Connectez-vous à un serveur de corrections NTRIP.

5. Effectuez vos relevés.

</Steps>
