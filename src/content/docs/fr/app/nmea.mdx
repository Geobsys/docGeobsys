Ce guide explique comment configurer les messages NMEA émis par le récepteur GEOSTIX via Bluetooth à l'aide de l'application GEOSTIX APP.

<Aside type="tip" title="Prérequis">
- Un récepteur GEOSTIX connecté via l'application GEOSTIX APP
- Le mode Expert activé dans les paramètres de l'application
</Aside>

## Activer le mode Expert

<Steps>

1. **Ouvrez l'onglet Paramètres.**

    Dans l'application GEOSTIX APP, accédez à l'onglet **Paramètres**.

2. **Activez le mode Expert.**

    Basculez l'interrupteur **Mode expert** en bas de l'écran. Cela déverrouille les options de configuration avancées, notamment les paramètres de sortie NMEA.

    <div style="display: flex; justify-content: center; gap: 2rem;">
        <img src="/images/nmea/settings_fr.png" width="300" />
    </div>

</Steps>

## Configurer les fréquences de sortie NMEA

<Steps>

1. **Connectez-vous à votre GEOSTIX.**

    Depuis l'application, connectez-vous à votre récepteur GEOSTIX. Consultez le [guide de démarrage rapide](/fr/geostix/quick_guide/) pour plus de détails.

2. **Ouvrez l'écran Sortie NMEA.**

    Depuis l'écran **Récepteur**, appuyez sur **Sortie NMEA**. Cet écran liste tous les messages NMEA configurables et leur fréquence d'émission actuelle.

    <div style="display: flex; justify-content: center; gap: 2rem;">
        <img src="/images/nmea/nmea_list_fr.png" width="300" />
    </div>

3. **Ajustez la fréquence pour chaque message.**

    Appuyez sur la valeur de fréquence à côté de chaque message pour la modifier. Les fréquences disponibles sont : **Désactivé**, **5 Hz**, **1 Hz**, **0,5 Hz** et **0,1 Hz**.

    <div style="display: flex; justify-content: center; gap: 2rem;">
        <img src="/images/nmea/nmea_rates_fr.png" width="300" />
    </div>

4. **Réinitialiser les valeurs par défaut (optionnel).**

    Appuyez sur **Réinitialiser par défaut** en bas de l'écran pour restaurer la configuration NMEA d'usine.

</Steps>

<Aside type="note" title="Référence des messages NMEA">
Pour une description de chaque message NMEA et les exigences minimales pour le levé topographique, consultez la page [Sortie NMEA](/fr/geostix/nmea/).
</Aside>
